# Week 7 - AI Prompts Log

## Prompt 1

### Prompt
Analyze the following hotel review.

Return:
- Summary
- Sentiment
- Key Points
- Professional Reply

### Example Input
The room was clean and staff was friendly.

### Example Output
Summary:
Guest had a pleasant stay.

Sentiment:
Positive

Key Points:
- Clean room
- Friendly staff

Professional Reply:
Thank you for your valuable feedback.

---

## Prompt 2

### Prompt
You are an AI assistant for hotel review analysis.

Analyze the review and provide:

Summary

Sentiment

Key Points

Professional Reply

### Example Input
Food was delicious but room was noisy.

### Example Output

Summary:
Mixed experience.

Sentiment:
Neutral

Key Points:
- Good food
- Noisy room

Professional Reply:
Thank you for your feedback. We will improve the room experience.

---

## Prompt 3 (Best)

### Prompt

You are an AI assistant for hotel review analysis.

Return ONLY:

Summary:
Sentiment:
Key Points:
Professional Reply:

### Example Input

The room was very clean and staff was friendly.

### Example Output

Summary:
Excellent stay.

Sentiment:
Positive

Key Points:
- Clean room
- Friendly staff

Professional Reply:
Thank you for staying with us.

---

## Best Prompt

Prompt 3 performed the best because it returned structured, consistent, and easy-to-read responses. It clearly separated the summary, sentiment, key points, and professional reply, making it suitable for display in the application dashboard. The responses were concise, accurate, and required minimal post-processing.
